JSP 구성 요소

1. JSP 지시자, JSP Directive
	- <%@ %>
	- JSP Container(톰캣)에게 페이지 처리를 위한 여러가지 상황을 전달server
	a. page 지시자
	b. include 지시자
	c. taglib 지시자

2. 스크립트 요소, Scripting Elements
	- <% %>
	- 자바 코드와 관련된 기능들
	a. 스크립틀릿, Scriptlet
	b. 익스프레션, Expression
	c. 선언부, Declaration 

3. 액션 태그, Action Tags
	- <jsp:태그명>, <c:태그명> 등..
	- 프로그래밍 기능이 있는 태그

4. 내장 객체, Built-in Objects
	- 톰캣이 알아서 생성 후 개발자에게 제공하는 객체

page 지시자
- webapp > "ex03.jsp"
- JSP 페이지 실행과 관련된 여러가지 정보를 톰캣에게 알려주는 역할
- JSP 페이지 첫줄에 작성한다.
- page 지시자는 여러개 작성이 가능하다.

- <%@ page language="java" contentType="text/html; charset=UTF-8"
    pageEncoding="UTF-8"%>
    a. language="java"
    	- JSP 페이지에서 사용할 프로그래밍 언어 지정
    	
    b. contentType="text/html; charset=UTF-8"
    	- 동적으로 생성된 페이지의 설정
    	- text/html: MIME 타입
    	- charset=UTF-8: 돌려줄 HTML 페이지의 인코딩
    
    c. pageEncoding="UTF-8"%
    	- 현재 JSP 페이지의 설정
