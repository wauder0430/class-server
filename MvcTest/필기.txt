Servlet
- 장점: 자바 기반
- 단점: 클라이언트 코드 불편 


JSP
- 장점: HTML 기반
- 단점: 자바 불편


Servlet + JSP > 자기가 잘하는걸 하자!!
- 자바 코드 > Servlet에서 작성
- 클라이언트 코드 > JSP에서 작성
- 단점 > 사람 할일 증가



페이지 작업
1. 브라우저가 서블릿 호출한다.(JSP 호출 X)
2. 호출된 서블릿 > 자바 작업
3. 서블릿의 업무 완료 > (데이터 전달)? > JSP 호출
4. 호출된 JSP > HTML 문서 작업
5. HTML 문서를 브라우저에게 전달한다.

JSP Model 1 > 자바 코드 + 클라이언트 코드 > 1장의 JSP 페이지에 작성하는 방식
JSP Model 2 > 서블릿(자바) + JSP(클라이언트) > MVC 디자인 패턴 차용
